## 使用预编译的可执行文件

### Windows 系统

打开 <https://github.com/llx07/Bookstore-2025/releases/latest> ，下载 `server-build-windows.zip` 压缩包，解压后双击 `server.exe` 即可运行。

### Linux 系统
打开 <https://github.com/llx07/Bookstore-2025/releases/latest> ，下载 `server-build-ubuntu.tar.gz` 压缩包，解压后执行 `./server` 即可运行。

## 编译源代码运行
本项目提供了自动编译脚本 `build_server.sh / build_server.ps1`，可以在不同系统下使用该脚本进行编译。

### Windows 系统
#### 前置依赖
- Msys2 环境。
- Mingw64
- OpenSSL
- Cmake 3.16 及以上版本。
- Git
#### 获取源代码
```bash
git clone https://github.com/llx07/Bookstore-2025.git .
```
#### 编译
```bash
./build_server.ps1
```

### Linux 系统
#### 前置依赖
- C++20 支持的编译器
- CMake 3.16 及以上版本。
- Git
- OpenSSL
#### 获取源代码
```bash
git clone https://github.com/llx07/Bookstore-2025.git .
```
#### 编译
```bash
chmod +x ./build_server.sh
./build_server.sh
```
